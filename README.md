# 🚢 Titanic Survival Analysis - AnalystLab Africa Internship

## 📌 Project Overview / Présentation du projet
This project is part of my Machine Learning internship. The goal is to perform data preprocessing and Exploratory Data Analysis (EDA) on the famous Titanic dataset.
Ce projet fait partie de mon stage en Machine Learning. L'objectif est d'effectuer le prétraitement des données et l'analyse exploratoire (EDA) sur le célèbre dataset du Titanic.

## 🛠️ Tasks Performed / Tâches réalisées
* **Data Cleaning:** Handled missing values (Age, Embarked) and removed unnecessary columns (Cabin, Name, Ticket).
* **Nettoyage des données :** Gestion des valeurs manquantes (Age, Embarked) et suppression des colonnes inutiles.
* **Feature Encoding:** Converted categorical data (Sex, Embarked) into numerical format.
* **Encodage des variables :** Conversion des données catégorielles en format numérique.
* **Data Scaling:** Applied StandardScaler to normalize Age and Fare.
* **Mise à l'échelle :** Application du StandardScaler pour normaliser l'âge et le prix du billet.
* **EDA:** Created visualizations to understand correlations between survival and features (Class, Sex).
* **EDA :** Création de visualisations pour comprendre les corrélations avec la survie.

## 📊 Key Insights / Conclusions Clés
- **Sex:** Females had a significantly higher survival rate (Correlation: 0.54).
- **Sexe :** Les femmes avaient un taux de survie nettement plus élevé.
- **Pclass:** 1st class passengers had better chances of survival than 3rd class.
- **Classe :** Les passagers de 1ère classe avaient plus de chances de survivre.
